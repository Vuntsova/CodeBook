# CodeBook

CodeBook is Facebook for junior webdevelopers who want to share their experiance learning diferent programs and learning  from other junior coders experiance. Users will be able to:

- log in
- log out
- create a profile
- post
- edit post
- delete post
- comment
- like
- dislike posts
